## 📖 About The Project

Hangman Game is a fun and interactive console-based word guessing game built using Python. The objective of the game is to correctly identify a hidden word by guessing one letter at a time.

Players are given a limited number of incorrect attempts. Each wrong guess brings the player closer to losing the game, while correct guesses gradually reveal the hidden word. The project demonstrates the use of Python programming concepts such as loops, conditional statements, strings, lists, and user interaction.

This project is ideal for beginners learning Python and basic game development concepts.

## 🧠 Game Workflow

### 1️⃣ Start Game
- Random word is selected
- Hidden word is displayed using underscores

### 2️⃣ Player Guess
- User enters a letter
- System validates the input

### 3️⃣ Check Guess
- Correct guess reveals letters
- Incorrect guess reduces attempts

### 4️⃣ Update Progress
- Display current word status
- Show remaining attempts

### 5️⃣ Game Result
- Player wins by guessing the word
- Player loses if attempts reach zero
